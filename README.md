# HarbourWallpaperSwitcher 
多平台壁纸切换工具，让战舰少女R港口景色在你的设备上栩栩如生
Wallpaper switcher for Warship Girls R harbor scenes, compatible with multiple platforms. 

## Features
* 多平台支持
  已支持安卓（Termux）、Windows、Ubuntu（Gnome）等平台
* 基于Python
* 精确计算季节与天色变化，切换壁纸
* 采用GPS（仅安卓Termux支持）
* 占用资源低

> 安卓（Termux）和Windows目前请移步[WarshipGirlsR-tools](https://github.com/Octopustank/WarshipGirlsR-tools)。未来将迁出合并
